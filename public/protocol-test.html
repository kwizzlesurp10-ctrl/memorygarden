🌱 MemoryGarden Protocol Handlers

Test protocol handlers and deep linking

⚠️ Important: Protocol handlers only work when MemoryGarden is installed as a Progressive Web App (PWA). Please install the app first, then return to this page to test the protocol links.

🌿 Plant a Memory

Opens the plant memory modal in MemoryGarden.

Plant Memory

web+plantmemory://

👁️ View Specific Memory

Opens a specific memory by ID. Enter a memory ID to test:



web+viewmemory://<memory-id>

🏡 General Protocol

Invokes the general MemoryGarden protocol handler.

Test General Protocol

web+memorygarden://<action>
